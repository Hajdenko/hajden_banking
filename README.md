# hajden_banking
Advanced Banking System with built in billing and robbery system.

## <b>Requirements</b>
- ox_lib
- oxmysql
- ox_inventory
- ESX (for now)

## <b>Features</b>
- 0.0ms Usage
  
Usage
- Highly Configurable
- Easy to Use
- Free
- Protected
- PIN when using ATM
- Bank/ATM dependent banking menu
- No deposit in ATMs (if the ATM is close to a bank, you can deposit)
- Quick Actions to deposit and withdraw
- Transfer money to another user
- Billing menu (Personal Bills, Job Bills)
- Only users with a job can send a bill
- Commands to open the billing menu
- Robbery hacking minigame
- You can access the vault using ID Card
- Smooth vault opening
- Cash Grab animation

## <b>Installation</b>
- Download hajden_banking
- Add hajden_banking into your resource folder
- Modify the config
- If you want to use the robbery script, you need to create new items in your ox_inventory
- Add hajden_banking into your server.cfg file or ensure hajden_banking
- Restart server
